pgmq4s.PgmqAdmin
trait PgmqAdmin[F[_]] extends PgmqAdminBackend[F]

Tagless-final algebra for PGMQ queue management and observability.

Provides create, drop, purge, metrics, and listing operations. Each database backend supplies a concrete implementation (e.g. DoobiePgmqAdmin, SkunkPgmqAdmin).
