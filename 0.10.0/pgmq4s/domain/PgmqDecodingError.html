PgmqDecodingError

pgmq4s.domain.PgmqDecodingError
class PgmqDecodingError(val messageId: MessageId, val queue: QueueName, cause: Throwable) extends RuntimeException, NoStackTrace

Raised when a raw message cannot be decoded into the expected payload or header types.

The original decode error is available via getCause. No raw JSON is included to avoid leaking sensitive data into logs.
